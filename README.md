## :wrench: Install and run

Open terminal:

```sh
# Clone this repo
git clone https://github.com/osvaldokalvaitir/reactjs-star-wars

# Entry in folder
cd reactjs-star-wars

# Install deps with npm or yarn
npm install | yarn

# Launch the app with npm or yarn
npm run start | yarn start
```
